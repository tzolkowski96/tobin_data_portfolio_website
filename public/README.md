# UFood Customer Analysis

This folder contains a comprehensive analysis of UFood's customer data and marketing campaign performance. UFood is a leading food delivery app in Brazil, present in over a thousand cities. The goal of this analysis is to understand customer behavior, evaluate marketing campaign effectiveness, and provide data-driven insights to improve UFood's marketing strategies.

## Files

- `ufood_customer_analysis.html`: An HTML version of the analysis notebook, suitable for viewing in a web browser.
- `ufood_customer_analysis.ipynb`: The original Jupyter Notebook containing the Python code and detailed explanations of the analysis. 
- `ufood_customer_analysis.pdf`: A PDF version of the analysis notebook, suitable for printing or sharing.

## Dataset

The dataset used in this analysis, `u_food_marketing.csv`, is provided by AnalystBuilder as part of its Pandas for Data Analysis course's Food Marketing Data Analysis Project. It contains information about UFood's customers, their purchasing behavior, and their responses to various marketing campaigns.

## Analysis Overview

The analysis covers several key areas:

1. Exploratory Data Analysis
2. Customer Segmentation
3. Campaign Performance Analysis
4. Product Preference Analysis
5. Time-based Customer Behavior Analysis
6. A/B Testing of Marketing Campaigns

Various data science techniques were applied, including:

- Statistical analysis and visualization with Pandas, Matplotlib, and Seaborn
- K-means clustering for customer segmentation
- Cohort analysis to understand customer behavior over time
- Hypothesis testing to compare campaign effectiveness

## Key Findings

- Identified three distinct customer segments with unique characteristics and preferences
- Uncovered significant differences in campaign effectiveness, with insights for optimization
- Revealed product affinity patterns that can inform cross-selling and bundling strategies
- Tracked customer value evolution over time, identifying key intervention points
- Validated the superiority of a specific campaign through rigorous A/B testing

## Recommendations

Based on the analysis, several strategic recommendations were made to improve UFood's marketing performance, including:

1. Tailoring marketing strategies to each customer segment
2. Focusing on high-performing campaigns and analyzing their success factors
3. Leveraging product affinity insights for targeted promotions and bundles
4. Implementing customer lifecycle management strategies based on tenure analysis
5. Optimizing the website experience based on customer engagement patterns

For detailed findings and insights, please refer to the analysis notebooks.
