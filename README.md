# Website Redirect

This is a simple redirect page that automatically forwards visitors to Tobin Zolkowski's new portfolio location.

## Features

- 5-second countdown timer
- Automatic redirect to new location
- "Go now" link for immediate redirection
- Fallback for users with JavaScript disabled
- Responsive design with gradient background
- Accessible and user-friendly

## New Location

The portfolio has moved to: https://tzolkowski96.github.io/portfolio/

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Built with React, TypeScript, and Tailwind CSS.
